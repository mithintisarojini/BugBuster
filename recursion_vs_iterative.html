Some problems can be solved using either recursion or iteration.

Recursive Approach:

  • Code is usually simpler and easier to understand.
  • May consume more memory due to call stack.

Iterative Approach:

  • Uses loops instead of function calls.
  • More memory-efficient and avoids stack overflow.

Example (Factorial in Iterative Way):

int factorial(int n) {
    int result = 1;
    for(int i=1; i<=n; i++) result *= i;
    return result;
}
