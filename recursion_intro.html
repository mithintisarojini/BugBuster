What is Recursion?

Recursion is a programming technique where a function calls itself directly or indirectly to solve a problem. Every recursive function requires a base case to terminate.

Why Use Recursion?

  • Simplifies code for problems like factorial, Fibonacci, and tree traversals.
  • Useful in divide-and-conquer algorithms.
  • Helps in exploring all possibilities in backtracking problems.

Example: Factorial Function (JavaScript)

function factorial(n) {
    if (n === 0 || n === 1) return 1;
    return n * factorial(n - 1);
}

console.log(factorial(5)); // Output: 120

Key Points

  • Every recursive function needs a base case to avoid infinite recursion.
  • Recursive functions can often be converted into iterative ones.
  • Understand the call stack to debug recursion effectively.
