Efficient array processing by using two indexes.

What is Two-Pointer?

The two-pointer technique uses two indices that move across the array depending on certain conditions. It helps solve problems faster than brute-force approaches.

When to Use?

  • Searching for pairs/triplets that satisfy conditions.
  • Finding subarrays with certain properties.
  • Sorting and merging operations.

Example: Pair Sum

int arr[] = {1, 3, 4, 5, 7, 10, 11};
int target = 9;
int i = 0, j = 6;

while(i < j) {
    if(arr[i] + arr[j] == target) {
        cout << arr[i] << ", " << arr[j];
        break;
    } else if(arr[i] + arr[j] < target) {
        i++;
    } else {
        j--;
    }
}

O(n), much better than O(nÂ²) brute force.
